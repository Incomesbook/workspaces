# S10_GitHub / workspaces

Created: 2026-05-03.
Canonical location: `J:\Setup_VcCode_Workspace\S10_GitHub\workspaces`.

This repository is a seed for workspace structure, scripts, manifests and small configuration files. It intentionally excludes bulky runtime caches such as `_LIVE`, `_COLLECTED_FS`, `workspaceStorage`, browser caches and raw `*.jsonl` chat streams.

Large archives, videos, raw AI runtime state, `.env` files and API-key files must not be pushed to GitHub. Keep them in local backup/storage workflows with explicit secret policy.

# AI Chat Local Raw Mirror Policy

Mode: POLICY ONLY.

## Allowed

- Copy existing AI chat/application roots to J:\_AI_CHATS_ОБЩИЕ\_SOURCES.
- Keep originals in place.
- Keep backup of destination files before overwrite.
- Commit only manifests/status/control docs to GitHub.

## Blocked

- Delete originals.
- Move originals.
- Raw chat GitHub push.
- Git LFS.
- Archive/password workflow.
- ENV/PATH edits.
- Blind overwrite without backup.

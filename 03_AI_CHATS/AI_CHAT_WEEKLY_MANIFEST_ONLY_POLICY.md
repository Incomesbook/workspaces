# AI Chat Weekly Manifest-Only Policy

Mode: POLICY ONLY.

## Weekly sync scope

Allowed:
- manifest docs
- CSV metadata
- status docs
- restore maps
- control docs

Not allowed:
- raw chat histories
- .jsonl raw files
- databases
- archives
- secrets
- Git LFS
- deletion
- original movement

## Rule

Weekly automation must stay manifest-only unless Igor separately approves raw chat sync.
